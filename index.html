Technology Stack

A cloud native technology stack based on containers with runtimes, development frameworks, DevOps and GitOps Tooling.

Our Technology Stack

The project is built on a fully cloud-native, open-source technology stack designed for scalability, security, and rapid delivery.

The frontend combines TypeScript, React, and Tailwind to provide a modern, responsive user experience, while the backend leverages NestJS and Quarkus to deliver high-performance microservices.

PostgreSQL ensures reliable data persistence, and Keycloak provides robust identity and access management.

The platform runs on OpenShift, with GitHub as the development hub. ArgoCD and Tekton enable GitOps-driven continuous delivery and pipeline automation, ensuring consistent, automated, and secure deployments across environments.
